# AI Coding Tool Enhancement PRD

## Project Overview
Enhance the ai-coding-assistants-setup project with improved VS Code integration, expanded MCP support, better contributor experience, and support for non-JavaScript repositories.

## Key Requirements

### 1. Add or Integrate More AI Tools
- Add detection for existing GitHub Copilot installations
- Provide option to disable/override Copilot
- Update documentation to clarify differences between Claude, Roo, and Copilot

### 2. Expand Model Context Protocols (MCPs)
- Add StackOverflow Search MCP
- Add Command Shell MCP
- Add Database Query MCP
- Add Embedding Search (Weaviate/Chroma) MCP
- Update configuration files and documentation accordingly

### 3. Improve Contributor Friendliness
- Modularize the CLI script into smaller components
- Add prerequisites to README
- Enhance CONTRIBUTING.md
- Add CI with GitHub Actions
- Improve onboarding documentation

### 4. Add Basic Tests
- Add snapshot or functional tests for config generation
- Create test fixtures and mocks
- Integrate with existing build process

### 5. Document MCP Servers
- Create a comprehensive table of all supported MCPs
- Include purpose and requirements for each MCP
- Integrate into documentation

### 6. Automate Claude Hooks for Commits
- Add Git hook support using Husky or lint-staged
- Configure pre-commit hooks to run Claude for code review
- Add optional configuration for review strictness
- Explore additional AI CLI hooks

### 7. Add Support for Non-JavaScript Repositories
- Modify project root detection to work without package.json
- Implement graceful fallbacks for key functions that rely on package.json
- Provide clear messaging about limited functionality in non-JavaScript repositories
- Update error handling to be more informative
- Add basic CLI flags for specifying project type
- Add language detection for Python, Ruby, Shell, and other common languages
- Update documentation with information about non-JavaScript repository support

### 8. Optional Future Features
- Auto-run Playwright tests after AI-generated code
- Commands to reformat project-wide settings
- AI onboarding script
- Enhanced language-specific support for non-JavaScript repositories
